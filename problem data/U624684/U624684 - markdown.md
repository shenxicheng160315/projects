# U624684 坤坤的高考成绩

## 题目背景

坤坤的高考成绩炉了，但是因为数学不好，算不出来平均分，于是他找到了你，请你来算算坤坤的平均分并算出他能上上哪个学校。

## 题目描述

输入数学、语文、英语三项的成绩，输出平均分和学校。

## 输入格式

共一行，第一行输入数学、语文、英语三项的成绩并用空格隔开。

## 输出格式

共两行，第一行输出平均分，第二行输出学校。

## 输入输出样例 #1

### 输入 #1

```
100 100 100
```

### 输出 #1

```
100
学校1
```

## 说明/提示

80分以上可以上学校1                                   
50分以上可以上学校2  
30分以上可以上学校3  
30分以下可以上学校4  
0≤数学、语文、英语成绩≥100
